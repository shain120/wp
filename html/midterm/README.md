## 操縱方法
WASD/↑↓←→鍵操縱  
Esc 暫停 再按一下開始
## detal
1. 設定讓蘋果碰到身體也會得分
2. 讓Canvas新增像素格
3. 設定esc暫停 在按一次繼續
4. 隨著難度變更產生不同的變速
5. 如果長度為0時 可以前後左右 不受keyDown()規則
6. 遊戲開始時或重新開始按esc時不會 clear timeout
7. 更改顏色 (蛇 蘋果)
8. 增加難度 運用addEventListener 來做難度的變更
9. 禁用上下左右鍵去控制select
