## 操縱方法
WASD/↑↓←→鍵操縱  
Esc 暫停 再按一下開始
## detal
1. 設定讓蘋果碰到身體也會得分
2. 讓Canvas新增像素格
3. 設定esc暫停 在按一次繼續
4. 變速 每吃兩個加一速度 分數到達15時加3速度
5. 如果長度為0時 可以前後左右 不受keyDown()規則
6. 遊戲開始時或重新開始按esc時不會 clear timeout
7. 快速按兩個鍵可能導致遊戲結束以解決，但快速按兩個鍵會讓頭穿過身體
8. 更改顏色 (蛇 蘋果)
